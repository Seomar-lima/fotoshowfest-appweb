INSTRUÇÕES PARA O USO:

1. Coloque o arquivo service-worker.js na raiz do seu projeto (ao lado do index.html).
2. Coloque o manifest.json também na raiz.
3. No index.html, certifique-se de ter:
    <link rel="manifest" href="manifest.json" />
    e o script de registro do service worker.
4. Inclua o screenshot1.jpg e os ícones na raiz conforme indicado no manifest.
